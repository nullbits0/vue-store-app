# Vue Store App

Aplicación web de e-commerce desarrollada con Vue.js 3.
Permite gestionar productos y usuarios mediante una API REST.

## Funcionalidades
- Autenticación de usuarios con validación de credenciales
- CRUD completo de productos (crear, editar, eliminar)
- CRUD completo de usuarios
- Navegación entre vistas con Vue Router
- Consumo de API REST con Axios

## Tecnologías
- Vue.js 3
- Vue Router
- Axios
- MockAPI (REST API)
- JavaScript (ES6+)

## Cómo ejecutarlo
npm install
npm run dev
